## How to use this file
When `/workflow init` is run, Architect reads this file and follows its
instructions to create the `llm-wiki/` folder structure, including:
- `llm-wiki/raw/sources/`
- `llm-wiki/raw/designs/`
- `llm-wiki/raw/history/`
- `llm-wiki/wiki/index.md`
- `llm-wiki/wiki/architecture/`
- `llm-wiki/wiki/decisions/`
- `llm-wiki/wiki/pitfalls/`

And generates the wiki interaction rules as a skill saved to:
`.ai/skills/wiki_rules.md`
