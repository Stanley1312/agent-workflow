## Vision
Enable Claude Code to analyze local images through a local Ollama-powered vision MCP server using a configurable local vision model.

**Goal:** Establish the AI development workflow foundation for the repository.
**Status:** Done
**Contains:** Workflow initialization

## Milestone 2: Local Vision MCP Foundation
**Goal:** Make Claude Code capable of sending a local image to an MCP server and receiving analysis from a configured Ollama vision model.
**Status:** Pending
**Contains:** Vision MCP server for Ollama; Image input support for Claude Code workflows; Ollama connectivity and model validation; Environment-configurable Ollama vision model

## Milestone 3: Practical Claude Code Usage
**Goal:** Make the MCP server reliable and usable for real Claude Code image analysis workflows.
**Status:** Pending
**Contains:** Configurable analysis prompts; Installation and Claude Code configuration guide; Robust error handling for image analysis requests

## Milestone 4: Advanced Image Workflows
**Goal:** Expand the MCP server beyond single-image ad hoc analysis into richer reusable workflows.
**Status:** Pending
**Contains:** Multiple image analysis support; Reusable prompt presets
